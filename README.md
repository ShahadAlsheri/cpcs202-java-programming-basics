# CPCS 202 – Java Programming Assignments

This repository contains university assignments developed for the course CPCS 202 (Programming 1).  
All projects are implemented in Java and cover fundamental programming concepts.

## Assignments Overview

### Assignment 2
1. **ass2Q1.java** – Determines the quadrant or axis of a point based on X and Y coordinates.  
2. **ass2Q2.java** – Sorts three numbers in ascending order.  
3. **ass2Q3.java** – Calculates total price based on menu selection and quantity.

### Assignment 3
1. **ass3Q1.java** – Prints sequences using loops.  
2. **ass3Q2.java** – Finds the alphabetical position of a letter.  
3. **ass3Q3.java** – Counts votes for Alcohol, Gasoline, Diesel until user stops input.  
4. **ass3Q4.java** – Tracks total and percentage of different animals in a survey.

## Topics Covered
- Conditional statements (if, switch)  
- Loops (for, while, do-while)  
- Methods  
- Basic input/output  
- Arrays and simple calculations

## Language
- Java
